Máy Tính Cơ Bản (Calculator Desktop)
Một ứng dụng máy tính bỏ túi đơn giản được xây dựng bằng Java Swing, hỗ trợ các phép tính cơ bản với giao diện trực quan và dễ sử dụng.

Yêu cầu 🔧
Java 11 hoặc phiên bản cao hơn.

Cài đặt 🔌
Nhấn nút Fork (góc trên bên phải trang GitHub) để lưu một bản sao của dự án vào tài khoản GitHub của bạn.

Tải mã nguồn dự án từ GitHub hoặc sử dụng lệnh sau để sao chép repository: git clone https://github.com/TranDangNgocCong/calculator_coban_Desktop.git

Mở dự án bằng IntelliJ IDEA hoặc bất kỳ IDE Java nào khác như Eclipse, NetBeans,...
Chạy ứng dụng và bắt đầu sử dụng.

Đóng góp 💡
Nếu bạn muốn đóng góp để cải thiện dự án bằng các ý tưởng hoặc tính năng mới, mọi Pull Request đều được hoan nghênh.

Nếu phát hiện lỗi hoặc gặp sự cố trong quá trình sử dụng, vui lòng tạo một Issue trong repository để chúng tôi có thể xem xét và khắc phục.

Xin cảm ơn vì sự quan tâm và đóng góp của bạn!
